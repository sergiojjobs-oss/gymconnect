# GymConnect

Plataforma web para conectar entrenadores personales con sus clientes.

## Funcionalidades

- Perfil público del entrenador (bio, servicios, precio)
- Búsqueda y selección de entrenador por parte del cliente
- Pago via PayPal para iniciar la relación entrenador-cliente
- Chat en tiempo real entre entrenador y cliente
- Publicación de rutinas y tareas por parte del entrenador

## Tecnologías

- **Frontend:** HTML + CSS + JavaScript (responsive)
- **Backend:** Java + Spring Boot
- **Base de datos:** PostgreSQL
- **Pagos:** PayPal SDK
- **Tiempo real:** WebSockets
